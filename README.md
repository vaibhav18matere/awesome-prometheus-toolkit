
# Promotheus Alert Toolkit dashboard

### Run Locally

Clone the project

```bash
  git clone https://github.com/vaibhav18matere/awesome-prometheus-toolkit.git
```

Go to the project directory

```bash
  cd awesome-promotheus-toolkit
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start

```
